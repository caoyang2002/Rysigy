# 仅使用
下载.zip文件
1. 使用`安装此软件`生成一个reg配置文件
2. 双击`install.reg`文件配置右键菜单启动
# 使用方法
1. 首先在当前的目录中创建一个存储库
2. 在在文件/文件夹/背景右键 -> Rysigy版本控制 -> 选择你需要的功能 (每一个功能执行结束后都需要按空格或其他任意一个按键退出)
